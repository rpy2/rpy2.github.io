rpy2 - R in Python

on Pypi

About

rpy2 is an interface to R running embedded in a Python process.

The project is mature, stable, documented, and widely used (older documentation).

License

The same license as R: GPLv2 or greater.

Questions and issues

Consider having a look at the documentation. Otherwise questions should preferably be asked on the rpy mailing-list on SourceForge, or on StackOverflow. Bugs, or wishes, can also go to the Github page for the project.

Source code

Repository

The source code is also in a public repository on Github.

Easy install

pip

Released source packages are available on PyPi. Installing should be as easy* as

pip install rpy2
(*: except on Windows)

Docker

A Docker image is also available to "kick the tires" with the last build from master:

a Jupyter notebook (on port 8888): 
docker run --rm -p 8888:8888 \
		    rpy2/jupyter-ubuntu:master-19.10
an ipython console: 
docker run -it --rm \
		    rpy2/jupyter-ubuntu:master-19.10 ipython

More information about Docker images can be found in the repository rpy2-docker.
