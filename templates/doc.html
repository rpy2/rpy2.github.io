Easy install

pip

Released source packages are available on PyPi. Installing should be as eaasy as

pip install rpy2

Docker

A Docker image is also available to "kick the tires" with the last build from master:

a Jupyter notebook (on port 8888): 
docker run --rm -p 8888:8888 rpy2/jupyter-ubuntu:master-20.04
an ipython console: 
docker run -it --rm rpy2/jupyter-ubuntu:master-20.04 ipython

More information about Docker images can be found in the repository rpy2-docker.
